# Portfolio – SAIPRIYA K

## Overview
A high-performance, modern portfolio built using React, Vite, and Tailwind CSS. This project showcases professional experience, technical expertise, and projects with a strong focus on premium design, glassmorphism, and advanced interaction patterns.

## Tech Stack
* **Framework:** React + Vite
* **Styling:** Tailwind CSS (Custom Theme)
* **Animations:** Framer Motion (Transitions)
* **Interactions:** AOS (Scroll effects), Swiper.js (Carousel)
* **Icons:** Lucide-React

## Features
* **Premium Dark Theme:** Deep navy and slate palette with vibrant violet and green accents.
* **Glassmorphism:** Consistent use of backdrop-blur and border-white/5 for a modern layered look.
* **Dynamic Animations:** Floating orbs in Hero, animated timeline in Experience, and parallax-style coverflow for Projects.
* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop views.
* **SEO Optimized:** Semantic HTML with proper meta tags and heading structure.

## Structure
* `src/data/profile.json`: Centralized resume data.
* `src/sections/`: Modularized UI components for each resume section.
* `src/index.css`: Global design system and glassmorphism utilities.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Design Notes
The project strictly follows the resume data provided, ensuring no paraphrasing or additional content. The color palette and aesthetics are designed to feel premium and state-of-the-art.
